Antes de ejecutar el programa, ejecutar los siguientes comandos en la VM para crear la base de datos y la tabla:

CREATE DATABASE LISTA_NOMBRES;
USE LISTA_NOMBRES;
CREATE TABLE LISTA(NAME TEXT(50), APELLIDO TEXT(50), CONTRASEÑA TEXT(50), EMAIL TEXT(50), POSTAL TEXT(50));

+ Asegurarse de que la IP de la máquina virtual es cambiada por la apropiada, así como el nombre de usuario y
la contraseña.
